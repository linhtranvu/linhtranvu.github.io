WYSIWYG Editor for Desktop

App ship with a license key for WYSIWYG Dashboard Editor for Desktop, it means that you are not limited to the exclude feature to only phone, but also your PC. This is also a thank-you feature for your support .

How it work

This desktop application works as a browser, so just open, input your Admin url, License key and go. App will automatically load WYSIWYG Editor. Node-RED dashboard 2.25.0 above required
